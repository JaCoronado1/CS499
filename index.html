Enhancements Narrative

Software Design and Engineering Enhancement

To improve the structure and maintainability of my food inventory application, I applied software design and engineering principles to refactor and organize the codebase. I implemented the Model-View-Controller (MVC) architecture, separated responsibilities across components, and created reusable utility classes. This enhanced the clarity and scalability of the application, making it easier to maintain and expand.

Algorithms and Data Structures Enhancement

I optimized the data retrieval and filtering logic in the application by applying efficient algorithms. I introduced a HashMap for fast item lookup and improved the expiration check process using pre-sorted data and targeted comparisons. These changes significantly reduced processing time and improved user experience.

Database Enhancement

I redesigned the SQLite database schema by normalizing the structure and introducing foreign key constraints. This improved data integrity and eliminated redundancy. Separate tables for categories and units were created and linked via foreign keys, resulting in cleaner data management and faster queries.
